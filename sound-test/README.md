# Sound Test
Change sound effects and music in the game

## Getting Started
### Sound Effects
All sound effects are wav files, these are located in the ```resources/sfx``` folder

To add new sound effects you need to edit the ```resources/sounds.xml```

### Music
All music are ogg files, these are located in the ```resources/music``` folder

Just use ```resources/music.xml``` as a references to know how to name your files
